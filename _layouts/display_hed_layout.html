HED (Hierarchical Event Descriptor) is a structured vocabulary for annotating events.
HED and the HED schema have evolved over several generations; the current generation (HED-3G) HED base schema, beginning with version 8.0.0, is currently still under testing.
For more information, see hedtags.org.
